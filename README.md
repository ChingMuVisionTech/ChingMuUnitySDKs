### 1 引言

（1）ChingMu SDK for Unity 2.0插件提供了青瞳动捕在Unity中的各种功能API接口。<br>
（2）ChingMu SDK for Unity 2.0插件支持Unity2018.4.0f1及以上版本。

### 2 用途与功能

你可以使用ChingMU Unity插件在Unity中利用青瞳动捕提供的定位与动作捕捉来实现各种相关应用，如虚拟影视拍摄实时预览，虚拟角色直播，VR头显定位，或者无人机等物体的定位。
